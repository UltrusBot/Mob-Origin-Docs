# Consume Dye
Consumes a dye item and blends the color with the entity's Dyeable Model Color Power, if it has one.



### Example
```json
{
 "entity_action": {
     "type": "moborigins:consume_dye"
 }
}
```
Consumes a dye item and blends the color with the entity's Dyeable Model Color Power, if it has one.
